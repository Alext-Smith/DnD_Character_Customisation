# DnD Character Customisation and Stat Roller

A Python command-line toolkit for building and rolling 
stats for a DnD character from scratch.

## Files
- **Character Creator.py** — walks users through building 
  a full DnD character
- **Stats.py** — generates character stats using the 
  official four-dice-drop-lowest method

## Features
- Race, role, class and weapon selection
- Class descriptions and weapon proficiency by class
- Martial and simple weapon categorisation
- Stat rolling across all six core stats
- Displays individual rolls, dropped dice and final totals
- Input validation throughout

## How to run
1. Install Python 3.x
2. Open either file in any Python editor
3. Run the file and follow the prompts

## Built with
Python 3.13 | PyCharm

## Skills demonstrated
Functions, modular design, while loops, try/except,
dictionaries, lists, random number generation,
input validation, tuple unpacking
